使用支援／Support

從第一針到完成，都找得到下一步。

以下資料都保存在你的裝置上。若仍需要協助，請寄信給我們。

繁體中文

建立、編輯與完成作品

  1. 在「作品」右上角點 ＋ 建立作品並加入照片。
  2. 進入作品後點鉛筆修改名稱、照片與工具資料。
  3. 標記為已完成後，計數器與日記會鎖定；選擇恢復即可繼續。

六組計數器

輕點任一計數器即可加 1。長按可更改完整名稱、減 1 或歸零。織圖右側的彩色數字與作品中的六組計數器同步。

匯入與閱讀織圖

可從檔案選擇 PDF 或圖片。每份織圖會分別保存頁碼、高亮位置、手寫標記與每頁筆記。iPad 可全螢幕閱讀。

相機或照片無法使用

請前往系統「設定」→「隱私權與安全性」→「相機」或「照片」，允許 KnitNote 存取；App 只在你主動拍照或選取照片時使用權限。

備份與還原

在設定中選擇「匯出備份」，再把整個 .knitnote-backup 套件存到安全位置。還原時請選擇該備份套件本身，不要選擇裡面的個別檔案。

Apple Watch 同步

請確認 Watch 與安裝 KnitNote 的 iPhone 已配對。離線時的操作會保留在 Watch；重新連線後等待片刻即可同步。若顯示待同步，先保持兩端 App 開啟並確認藍牙與 Wi‑Fi 可用。

English

Create, edit, complete, and resume a project

  1. Tap ＋ in Projects to create a project and add a photo.
  2. Open the project and tap the pencil to edit its name, photo, or tool details.
  3. Completing a project locks counters and journal entries. Choose Resume to continue later.

Six counters

Tap any counter to add one. Touch and hold to rename it, subtract one, or reset it. Colored numbers beside a pattern stay synchronized with the same six project counters.

Import and read patterns

Select a PDF or image from Files. KnitNote saves each pattern's page, highlight, handwritten marks, and page notes independently. iPad supports a full-screen reading workspace.

Camera or photos are unavailable

Open system Settings → Privacy & Security → Camera or Photos and allow KnitNote access. The app uses permission only when you take or select a photo.

Backup and restore

Choose Export Backup in Settings and save the complete .knitnote-backup package. To restore, select the backup package itself—not an individual file inside it.

Apple Watch synchronization

Make sure the Watch is paired with the iPhone running KnitNote. Offline actions remain queued on Watch and synchronize after reconnection. If an item is pending, keep both apps open briefly and check Bluetooth and Wi‑Fi.
