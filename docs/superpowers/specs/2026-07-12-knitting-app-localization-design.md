# 毛線編織 App 多語系設計規格

日期：2026-07-12

## 1. 目標與範圍

App 必須完整支援多語系，而不只翻譯主要畫面。v1 提供繁體中文（`zh-Hant`）與英文（`en`），涵蓋所有 UI、首次使用導覽、說明內容、通知、無障礙文字、編織術語，以及 AI 功能中使用者看得到的提示與結果。

架構必須允許未來新增語言時，只增加翻譯資源與語言資料，不修改核心功能程式碼。英文是完整的基準語言；翻譯遺漏時回退到英文。

## 2. 使用者體驗

- 初次啟動時預設跟隨裝置語言；不支援的裝置語言使用英文。
- 使用者可在 App 設定中手動選擇「跟隨系統」、「繁體中文」或「English」。
- 手動選擇會保存在使用者設定中，並透過既有同步機制同步到其他裝置。
- 語言切換後，所有可更新畫面立即換語言；系統通知與小工具最遲於下一次重新載入時更新。
- 作品名稱、使用者筆記、匯入圖解等使用者內容不自動翻譯，避免改變原始資料。

## 3. 本地化架構

### 3.1 介面字串

所有顯示文字使用集中式本地化資源與具語意的穩定識別碼，不得直接寫死在畫面或商業邏輯中。字串資源需支援：

- 動態參數，例如目前排數與作品名稱。
- 單複數規則。
- 日期、時間、數字、貨幣與測量單位的地區格式。
- iPhone、iPad、Mac、Apple Watch、小工具、通知與無障礙標籤。
- 長文字、動態字級及未來由右至左語言的版面需求。

### 3.2 內容與說明

首次使用導覽、功能提示、空白狀態、錯誤說明、疑難排解、隱私說明及 AI 使用說明，使用可版本化的本地化內容資源。每一份內容需標記適用 App 版本，避免說明與介面脫節。

### 3.3 編織術語

編織術語使用獨立的結構化資料層，不與一般 UI 字串混用。每筆術語至少包含：

- 穩定術語 ID。
- 語言與地區。
- 標準名稱、常見縮寫及同義詞。
- 棒針或鉤針分類。
- 解釋與基本操作步驟。
- 美式／英式差異及對應術語。
- 可搜尋的別名。

切換語言後，術語名稱、解釋與搜尋索引一併更新。圖解中的原始縮寫應保留，並可並列顯示本地化解釋。

## 4. AI 本地化

AI 的系統指令、畫面提示、建議範例、輸出格式與免責說明均使用版本化的語言模板。呼叫 AI 時提供：

- 使用者選擇的回覆語言。
- 原始內容的語言（可判斷時）。
- 美式或英式術語偏好。
- 相關術語 ID 與核准翻譯。
- 不可改動的針數、排數、尺寸與單位。

AI 預設使用 App 選擇的語言回答，必要時並列原文術語。AI 結果必須標示為建議，並提醒使用者確認針數、尺寸與材料。模型無法可靠處理指定語言時，介面以本地化訊息告知並回退至英文結果。

## 5. 資料流與責任邊界

- 語言設定服務：決定有效語言、保存偏好與處理回退。
- UI 本地化層：提供畫面字串、格式與無障礙文字。
- 內容服務：提供對應語言及版本的導覽與說明。
- 術語服務：提供術語查詢、搜尋與地區差異。
- AI 提示組裝器：依語言、術語偏好與功能組合 AI 模板。

各單元只透過穩定 ID 溝通。畫面不得自行拼接句子，AI 模板不得成為 UI 翻譯來源，術語資料也不得混入一般介面字串。

## 6. 翻譯與發布流程

1. 開發先在英文基準資源建立字串 ID 與說明。
2. 繁體中文翻譯採用台灣常見用語。
3. 自動檢查缺漏、未使用字串、參數不一致及錯誤複數格式。
4. 一般內容由母語者審核；編織術語另由熟悉編織的人員校對。
5. App 商店文字、截圖、通知及說明內容與版本一同發布。
6. 未完成翻譯不得悄悄顯示字串 ID，必須安全回退至英文。

## 7. 測試策略

### 自動測試

- 有效語言選擇與英文回退。
- 每個基準字串在 v1 語言中的完整性。
- 動態參數、單複數與格式化結果。
- 語言切換後使用者資料保持不變。
- 術語跨語言搜尋及美式／英式對應。
- AI 提示使用正確語言、術語偏好與不可改動數值。

### 視覺與裝置測試

- iPhone、iPad、Mac 與 Apple Watch 的主要流程。
- 最小與最大支援畫面。
- 長字串、動態字級與無障礙朗讀。
- 通知、小工具、空白狀態、錯誤狀態及離線狀態。

### 使用者驗收

繁體中文與英文編織者需分別完成：建立作品、記錄排數、閱讀圖解、查詢術語、使用計算器及使用 AI 解釋編織步驟。

## 8. v1 驗收標準

- 繁體中文與英文可手動切換並可跟隨系統。
- 所有可見介面、導覽、說明、通知及無障礙文字均已本地化。
- 核心編織術語可用兩種語言搜尋與閱讀。
- AI 提示和回覆支援兩種語言並保留原始數值。
- 日期、數字與單位符合使用者地區設定。
- 主要裝置與大字模式無截斷、重疊或未翻譯內容。
- 新增語言不需修改核心功能程式碼。

## 9. v2 路線

v2 在不改變「三秒內記錄一排」核心體驗的前提下，依序加入：

1. 密度、起針、等距加減針、單位換算與毛線需求計算。
2. 完整毛線庫、庫存與材料缺口提醒。
3. PDF 標記、閱讀方向、鎖定與進階圖解高亮。
4. 繁中／英文語音計數與語音筆記。
5. 作品段落、獨立計數器、時間與溫和提醒。
6. AI 術語解釋、逐排整理、外文圖解輔助及毛線替代建議。
7. 新增日文、韓文與簡體中文，並支援原文／翻譯並列與翻譯修正回報。
8. 強化跨裝置同步、離線恢復、匯出與備份。

## 10. 非目標

- v1 不自動翻譯使用者作品名稱、筆記或匯入文件。
- v1 不承諾日文、韓文或簡體中文正式支援。
- AI 不取代使用者對針數、尺寸及圖解正確性的確認。
- 本功能不加入社群、商城、排行榜或廣告。
