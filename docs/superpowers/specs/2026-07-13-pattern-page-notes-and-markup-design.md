# 圖解每頁高亮、筆記與手寫標記設計

## 目標

補齊圖解閱讀第二階段，讓每一頁 PDF 可以保存自己的高亮位置、文字筆記與手寫標記，同時保留目前順暢的單頁閱讀及換頁行為。圖片圖解維持現有單頁閱讀狀態，但也可使用文字筆記與手寫標記。作品內與底部導覽列的圖解入口共用同一份資料，分別提供單一作品與全部作品的瀏覽方式。

## 實作順序

功能分為兩個可獨立驗證的階段：

1. 每頁高亮位置與文字筆記。
2. 每頁手寫標記。

第一階段通過資料升級、換頁與重新開啟測試後，才開始第二階段，避免同時改動過多閱讀器狀態。

## 兩個圖解入口

- 作品詳細畫面中的「圖解」是目前作品的快捷入口，只列出該作品所屬圖解。
- 底部導覽列的「圖解」是所有作品的圖解總覽，依作品分組顯示。
- 兩個入口直接讀取同一份 `PatternDocument`，不複製圖解檔案、筆記、高亮或手寫資料。
- 從任一入口點選圖解，都開啟同一個閱讀器並恢復相同閱讀頁及每頁狀態。
- 從底部總覽新增圖解時，使用者必須先選擇圖解所屬作品；從作品內新增時自動連結目前作品。
- 尚無圖解的作品仍可顯示於新增圖解的作品選擇器，但不需要在總覽中顯示空白分組。
- 刪除圖解會同步從兩個入口消失，且沿用相同的刪除確認與檔案清理流程。

## 每頁高亮位置

- 每頁保存橫向及縱向高亮位置，數值限制在 0 到 1。
- 高亮開關與模式（橫向、縱向、十字）由整份圖解共用。
- 離開目前頁面前保存位置，進入新頁面時載入該頁位置。
- 尚未保存位置的頁面，兩個方向都預設在中央。
- 舊圖解第一次升級時，將原有高亮位置保存到當時的閱讀頁；其他頁使用中央預設值。
- 圖片圖解只有一頁，沿用相同資料結構。

## 每頁文字筆記

- 每頁最多保存一則純文字筆記，可新增、修改或清空。
- 閱讀器工具列提供筆記按鈕，開啟簡單的編輯畫面。
- 有內容的頁面在筆記按鈕或頁碼附近顯示小型狀態標記。
- 筆記以頁碼作為索引，切換頁面、關閉閱讀器或 App 進入背景時保存。
- 清空內容等同刪除該頁筆記，不保留空白紀錄。
- 所有介面文字提供繁體中文與英文。

## 每頁手寫標記

- 使用者主動進入標記模式後，才攔截繪圖手勢；一般閱讀模式維持換頁及高亮拖動。
- 第一版支援自由畫筆、基本顏色、基本粗細、橡皮擦、復原及清除本頁。
- iPhone 與 iPad 支援手指及 Apple Pencil；Mac 支援滑鼠或觸控板。
- 每頁筆跡獨立保存，使用標準化頁面座標，裝置尺寸改變時仍對齊原位置。
- 筆跡資料獨立於原始 PDF 或圖片，不修改使用者匯入的檔案。
- 復原只作用於目前頁面當次的標記操作；清除本頁需要確認。

## 資料架構

每份 `PatternDocument` 新增以頁碼索引的頁面資料集合。每筆頁面資料包含：

- 橫向高亮位置。
- 縱向高亮位置。
- 可選文字筆記。
- 可選手寫筆跡參照或資料。

整份圖解繼續保存目前頁碼、高亮開關與高亮模式。儲存格式提升版本，解碼器必須接受舊版本並執行安全預設，不能要求使用者重新匯入圖解。

手寫資料與主要 JSON 紀錄分開保存於圖解專屬資料夾，以免大型筆跡讓主要資料檔持續膨脹。主要資料只保存筆跡檔案參照；刪除圖解時一併刪除其筆跡資料。

## 狀態流程

換頁時依序執行：

1. 將目前畫面中的兩個高亮位置及文字筆記寫回目前頁面狀態。
2. 切換 PDF 頁面。
3. 載入新頁面的高亮位置與筆記狀態。
4. 需要時載入新頁面的手寫筆跡。

PDF 閱讀器仍只負責頁面顯示與頁碼通知；頁面狀態切換由 SwiftUI 閱讀畫面協調，避免 PDFKit 同時修改資料模型造成循環更新。

## 錯誤處理

- 圖解載入失敗時，不覆蓋任何既有頁面資料。
- 主要資料保存失敗時顯示錯誤，並保留記憶體中的編輯內容供再次保存。
- 單頁筆跡檔案損壞時，只忽略該頁筆跡；高亮與文字筆記仍可使用。
- 頁碼超出 PDF 範圍時限制到有效頁碼，無效的頁面資料不影響開啟。
- 清除本頁筆跡必須由使用者確認，且不清除文字筆記或高亮位置。

## 測試與完成條件

- 不同頁面可保存不同的橫向與縱向位置，來回換頁後正確恢復。
- 高亮開關與模式切換頁面後保持共用。
- 每頁文字筆記可新增、修改、刪除，重新開啟後內容正確。
- 舊版資料可升級，原有閱讀頁與高亮位置不遺失。
- 手寫筆跡在不同視窗及裝置尺寸下仍對齊頁面內容。
- 標記模式與一般閱讀模式的手勢不互相干擾。
- PDF 關閉重開、App 進入背景、切換圖解及切換作品後狀態正確。
- 同一圖解從作品入口與底部總覽開啟時，頁碼、高亮、筆記及筆跡完全一致。
- 底部總覽依作品正確分組；新增圖解時必須連結有效作品。
- 刪除圖解後兩個入口都不再顯示，且不留下孤立檔案。
- iOS、iPadOS 與 macOS 皆能建置；核心資料與遷移測試全部通過。

## 不包含在本階段

- 圓圈、箭頭、直線等形狀工具。
- 將筆跡燒錄或匯出到原始 PDF。
- 多人共享與即時協作。
- AI 解析手寫內容或自動辨識圖解。
